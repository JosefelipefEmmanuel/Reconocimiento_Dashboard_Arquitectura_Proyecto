# 🔐 Sistema de Control de Acceso Biométrico con Reconocimiento Facial

Proyecto desarrollado como parte del curso de Arquitectura de Computadoras.

## 📌 Descripción

Sistema web de control de acceso que utiliza reconocimiento facial para autenticar usuarios y permitir o denegar el acceso a diferentes áreas (salas) según su rol.

Incluye integración simulada con dispositivos IoT (ESP32) para apertura de puertas.

---

## 🚀 Tecnologías utilizadas

- Node.js + Express
- MySQL
- Face-api.js
- TensorFlow.js
- HTML, CSS, JavaScript
- Bootstrap

---

## 🧠 Funcionalidades principales

- ✅ Reconocimiento facial en tiempo real
- ✅ Validación de usuarios desde base de datos
- ✅ Control de acceso por roles
- ✅ Dashboard administrativo
- ✅ Registro de accesos y fallos
- ✅ Simulación de integración con ESP32
- ✅ Sistema de permisos dinámico

---

## 🔒 Seguridad

- Autenticación basada en biometría facial
- Validación de roles (Admin / Usuario)
- Registro de intentos fallidos
